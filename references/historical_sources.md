# Historical sources

The site quotes its sources from page scans and public-domain transcriptions kept under `references/`. The heavy raw materials (page scans, OCR dumps, source PDFs, EEBO XML) are not committed to git, because every one of them can be downloaded again or regenerated by a script. This file records where each source comes from and how to restore it.

You rarely need any of this to read or build the site. Every quote in `corpus/` carries a `scanRef` link to the exact page it was transcribed from, so quotes can be verified in the browser. These materials matter only if you want to extend the corpus or re-run the ingest scripts.

## Christopher Cattan, *The Geomancie of Maister Christopher Cattan* (London, 1591)

The primary English source for the figure-in-house delineations and the technique chapters.

- Scan: Internet Archive item [b30337860](https://archive.org/details/b30337860) (Wellcome Library copy).
- `npm run fetch:cattan` downloads leaf images through the archive.org IIIF API into `references/scans/cattan/`.
- Transcriptions live in `references/extracted/cattan/` (committed) and are turned into `corpus/cattan/` by `scripts/ingest-cattan.mjs`.
- Note when citing: the leaf-to-printed-page offset drifts across the book (about -28 early, -34 late).

## Christopher Cattan, *La Géomance du Seigneur Christofe de Cattan* (Paris: G. Gilles, 1558)

The French original, kept for checking readings in the 1591 English translation.

- Scan: Internet Archive item [lageomancedvseig00catt](https://archive.org/details/lageomancedvseig00catt) (Duke University Libraries copy).
- Download the PDF and save it as `references/la_geomancie_de_cattan.pdf`.

## Pseudo-Agrippa, *Of Geomancy*, and Gerard of Cremona, *Astronomical Geomancy* (in *Fourth Book of Occult Philosophy*, trans. Robert Turner, London, 1655)

- Source text: EEBO-TCP transcription [A26563](https://github.com/textcreationpartnership/A26563), released into the public domain by the Text Creation Partnership. Use A26563, not A26562: only A26563 contains *Astronomical Geomancy*.
- `npm run fetch:eebo` downloads the TEI XML into `references/eebo/`.
- A few passages damaged in A26563 were supplied from the parallel transcription [A26562](https://github.com/textcreationpartnership/A26562).
- `scripts/tei-to-md.mjs` converts the XML into library pages.

## John Heydon, *Theomagia, or the Temple of Wisdome* (London, 1663)

A compilation built largely from Cattan and Agrippa (noted in the corpus editorial notes), quoted for its house delineations.

- Public scan: Internet Archive item [theomagiaortempl00heyd](https://archive.org/details/theomagiaortempl00heyd) (Getty Research Institute copy, freely downloadable).
- The scripts expect a 992-page PDF at the exact path `references/John Heydon - Theomagia, or the temple of wisdome (1663) - libgen.li.pdf`; the filename is hard-coded in `scripts/heydon-page.mjs`. A different scan may paginate slightly differently (the archive.org copy has 994 images), so OCR page numbers can shift by a page or two.
- `node scripts/heydon-page.mjs index` regenerates the rough per-page OCR text in `references/ocr/heydon/` with pdftotext. Use the OCR only to locate passages; transcribe from the page image (`node scripts/heydon-page.mjs image <page>` renders one).
- Transcriptions live in `references/extracted/heydon/` (committed) and are ingested by `scripts/ingest-heydon.mjs`.

## Aleister Crowley, "A Handbook of Geomancy" (*The Equinox*, vol. I no. 2, 1909)

A modern synthesis rather than a classic source; the site quotes it with editorial flags. Public domain.

- The extract used here is kept at `references/pdf/crowley-equinox-geomancy.pdf` (a standalone PDF of the handbook, available from [deceptionary.com](https://www.deceptionary.com/ftp/Geomancy-Crowley.pdf)).
- Full scans of *The Equinox* vol. I: Internet Archive item [crowley-equinox_202110](https://archive.org/details/crowley-equinox_202110). The handbook is in no. 2.
